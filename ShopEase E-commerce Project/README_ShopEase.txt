# ShopEase - Java E-commerce Application

ShopEase is a simple e-commerce desktop application built using Java Swing and MySQL for database management. It is designed as a group project by B.Tech Computer Science and Engineering students.

## 🔧 Technologies Used
- Java (OOP, Swing)
- MySQL (JDBC)
- MVC Architecture

## 📦 Features

### User Side:
- User registration and login
- Product browsing
- Order placement
- Order history view

### Admin Side:
- Admin login
- Product management (Add, Update, Delete)
- View all orders and order details

## 📁 Project Structure

```
src/
├── dao/               # Data Access Objects
├── models/            # POJO classes
├── ui/                # Swing UI screens
└── utils/             # Utility classes (e.g., DBConnection)
```

## ⚙️ Setup Instructions

1. Clone this repository or download the ZIP.
2. Import the project in your Java IDE (e.g., IntelliJ, Eclipse).
3. Set up the MySQL database using the `database.sql` script.
4. Update the database connection details in `utils/DBConnection.java`.
5. Run the `Main.java` file to start the application.
